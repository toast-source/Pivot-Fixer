# PNG 피봇 보정 툴

PNG 이미지의 피봇 위치를 맞추고, Aseprite 파일의 레이어 이동과 피봇 보정 복사본 생성을 도와주는 데스크톱 도구입니다.

주요 용도는 다음과 같습니다.

- PNG 이미지 피봇 보정, 변환, 일괄 처리
- 파일별 개별 설정 저장과 다중 선택 설정 적용
- Aseprite `.ase` / `.aseprite` 파일 미리보기
- Aseprite 레이어 이동
- Aseprite 피봇 분석과 피봇 보정 복사본 생성

## PNG 기능

- PNG 파일 또는 폴더를 드래그 앤 드롭으로 추가할 수 있습니다.
- 파일별로 개별 설정을 저장할 수 있습니다.
- 여러 파일을 선택한 뒤 현재 설정을 한 번에 적용할 수 있습니다.
- 세로/가로 오프셋을 조정할 수 있습니다.
- 실제 픽셀 영역의 좌우 bbox 중앙을 기준으로 정렬할 수 있습니다.
- 좌우 여백을 최소화할 수 있습니다.
- 가로 뒤집기, 세로 뒤집기를 지원합니다.
- 90도, 180도, 270도 회전을 지원합니다.
- 저장 방식은 원본 폴더 저장, 단일 폴더 취합, 원본 덮어쓰기를 선택할 수 있습니다.
- 마지막 작업 되돌리기를 지원합니다. 단, 원본 덮어쓰기 작업은 되돌리기 대상이 아닙니다.

## Aseprite 연동 도구

Aseprite 연동 도구에서는 `.ase` / `.aseprite` 파일을 추가해 레이어 이동이나 피봇 보정 복사본 생성을 할 수 있습니다.

- Aseprite.exe 경로를 자동 감지하거나 직접 설정할 수 있습니다.
- 파란색 선택은 현재 활성 파일, 미리보기 대상, 옵션 편집 대상입니다.
- 파란색 선택 파일은 단일 피봇 분석과 단일 피봇 복사본 생성 대상입니다.
- `☑` 체크는 레이어 이동 실행 대상이자, 체크된 파일 피봇 복사본 생성 대상입니다.

## 레이어와 태그 선택

- 레이어의 `보기` 체크는 미리보기 표시용입니다.
- 레이어의 `수정` 체크는 실제 레이어 이동과 피봇 보정 대상입니다.
- 레이어 목록의 `보기`, `수정` 헤더를 클릭하면 전체 체크/해제를 토글할 수 있습니다.
- 태그 체크는 `체크된 태그만` 모드에서 적용할 프레임 범위입니다.
- 태그를 클릭하면 해당 태그의 첫 프레임을 미리보기로 볼 수 있습니다.

## 레이어 이동 탭

- `가로이동`, `세로이동` 값을 실제 레이어 이동량으로 사용합니다.
- 레이어 이동 탭을 선택한 상태에서는 미리보기에서 이동 결과를 실시간으로 확인할 수 있습니다.
- 실행은 `☑` 체크된 Aseprite 파일 기준으로 진행됩니다.
- 새 파일로 저장하거나 원본 덮어쓰기를 선택할 수 있습니다.
- 원본 덮어쓰기는 `_aseprite_backup` 백업 후 진행되지만, 신중하게 사용해야 합니다.
- 이동 복사본 기본 접미사는 `_move`입니다.
- 접미사에 `{dx}`, `{dy}`를 직접 입력하면 이동값으로 치환됩니다.

## 피봇 보정 탭

- `가로이동`, `세로이동` 값은 직접 이동량이 아니라 피봇 기준 X/Y 오프셋입니다.
- `가로이동 보정 사용`, `세로이동 보정 사용`으로 X/Y 보정 방향을 선택할 수 있습니다.
- 피봇 보정 탭을 선택하면 빠른 예상 미리보기를 볼 수 있습니다.
- 빠른 예상 미리보기는 현재 `보기` 레이어 합성 이미지 기준입니다.
- 실제 저장은 `수정` 체크된 레이어 기준으로 정확하게 계산됩니다.
- 피봇 복사본 기본 접미사는 `_pivot`입니다.
- 접미사에 `{dx}`, `{dy}`를 직접 입력하면 피봇 오프셋 값으로 치환됩니다.

## 피봇 분석만 실행

- 실제 파일을 수정하지 않고 분석만 수행합니다.
- 프레임별 bbox, 이동값 dx/dy, 클리핑 여부를 확인할 수 있습니다.
- 태그별 첫 프레임 예상 미리보기를 제공합니다.
- 분석 결과 팝업에서는 기준선, 십자선, 확대/축소 기능을 사용할 수 있습니다.
- 피봇 보정 전후를 저장 전에 눈으로 확인하는 용도입니다.

## 피봇 보정 복사본 생성

- 기존 `피봇 보정 복사본 생성` 버튼은 현재 파란색 선택 파일 1개 기준입니다.
- 원본 파일은 수정하지 않고 복사본을 생성합니다.
- 클리핑이 예상되면 저장을 중단합니다.

## 체크된 파일 피봇 복사본 생성

`☑ 체크된 파일 피봇 복사본 생성`은 체크된 Aseprite 파일 전체를 대상으로 피봇 보정 복사본을 만듭니다.

- 실행 전 사전 점검 팝업이 표시됩니다.
- `설정 완료 파일만 실행`을 선택하면 저장된 설정이 충분한 파일만 처리합니다.
- `기본값 포함해서 실행`을 선택하면 설정이 부족한 파일도 기본값을 사용해 처리합니다.
- 설정이 부족한 파일은 `기본값 사용`으로 표시됩니다.
- 실패 예정 파일은 어떤 실행 방식에서도 실행하지 않습니다.
- 클리핑이 발생한 파일은 해당 파일만 건너뛰고 나머지는 계속 처리합니다.
- 결과 팝업에서 결과 폴더 열기, 성공 경로 복사, 전체 로그 복사를 사용할 수 있습니다.

## 안전성

- 피봇 보정 복사본 생성은 원본 파일을 수정하지 않습니다.
- 레이어 이동의 원본 덮어쓰기는 백업 후 진행되지만 주의가 필요합니다.
- 피봇 보정 중 클리핑이 감지되면 해당 복사본 저장을 중단합니다.
- 다중 Aseprite 파일 작업 전에는 사전 점검 팝업으로 설정 상태를 확인할 수 있습니다.

## 설치와 실행

1. Python 3.x를 설치합니다.
2. 프로젝트 폴더에서 필요한 패키지를 설치합니다.

```bash
pip install -r requirements.txt
```

3. `run.bat`이 준비되어 있다면 더블 클릭해서 실행할 수 있습니다.
4. 직접 실행하려면 다음 명령을 사용할 수 있습니다.

```bash
python main.py
```

Aseprite 연동 기능을 사용하려면 Aseprite.exe가 필요합니다. 앱에서 자동 감지하거나 설정 창에서 직접 경로를 지정할 수 있습니다.

## 현재 제한 사항

- Aseprite 작업에는 Undo/Redo가 아직 지원되지 않습니다.
- 피봇 빠른 미리보기는 `보기` 레이어 합성 이미지 기준 예상입니다. 실제 저장은 `수정` 레이어 기준으로 처리됩니다.
- 많은 Aseprite 파일을 한 번에 처리하면 Aseprite CLI 실행 때문에 시간이 걸릴 수 있습니다.
