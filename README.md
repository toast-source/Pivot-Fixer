# 🎨 PNG 피봇 보정 툴 (Pixel Art Optimizer)

이 프로그램은 게임 개발이나 애니메이션 작업 시, PNG 이미지(주로 픽셀 아트)의 **"피봇(중심점)"**을 쉽고 완벽하게 일괄 보정해 주는 데스크톱 애플리케이션입니다.

엔진에서 이미지를 띄울 때마다 캐릭터의 발이 땅에 파묻히거나 붕 뜨는 문제를 해결하기 위해, 이미지의 실제 픽셀을 분석하고 캔버스를 확장하여 수학적으로 완벽한 위치에 이미지를 재배치합니다.

## ✨ 핵심 기능

- **하단 픽셀 기준 세로 보정 (0~6px):** 
  - 단순히 이미지의 가운데를 맞추는 것이 아니라, **이미지의 가장 아래쪽(Bottom) 픽셀**을 캔버스의 중심(피봇) 기준선 바로 위에 얹습니다.
  - 사용자가 설정한 값(0~6px)만큼 픽셀이 피봇선 아래로 정확히 침범하도록 조절할 수 있습니다.
- **좌우 피봇 보정 (ON/OFF):**
  - 원본 이미지 여백에 상관없이 실제 눈에 보이는 픽셀들의 덩어리(알파 Bounding Box) 중앙을 최종 캔버스의 가로 중앙에 정확히 맞춥니다.
- **픽셀 아트 100% 무손실 보존:**
  - 결과물을 만들 때 캔버스 크기를 원본의 **무조건 2배**로 넓게 자동 확장합니다.
  - 이 과정에서 홀수/짝수 픽셀 연산 오차를 방지하여 0.5px 쏠림 현상을 원천 차단하며, 원본 픽셀 데이터가 잘리거나 흐려지는 일이 없습니다.
- **강력한 일괄 처리 (Batch) 및 덮어쓰기:**
  - 폴더 단위나 수십 장의 PNG를 드래그 앤 드롭으로 한 번에 불러올 수 있습니다.
  - 새로운 파일(`_pivotfix.png`)로 안전하게 저장하거나, 작업 속도를 위해 **원본 파일을 직접 덮어쓰는 기능**을 지원합니다.
- **안전한 실행 취소 (Undo):**
  - 실수로 처리를 눌렀더라도 '되돌리기' 버튼 한 번으로 직전에 생성된 파일들을 즉시 삭제(복구)할 수 있습니다. (단, 덮어쓰기 모드 제외)
- **픽셀 최적화 실시간 미리보기:**
  - 픽셀이 흐릿해지지 않도록 Nearest Neighbor 정수 배율 스케일링을 적용했습니다.
  - 체커보드 배경과 얇은 시안(Cyan)색 십자선을 통해 픽셀이 피봇에 어떻게 걸리는지 1px 단위로 명확하게 확인 가능합니다.

## 🚀 최근 업데이트 개선 사항
- **[UI/UX 전면 개편]** `ttk` 모던 테마(Clam) 적용 및 3단 분할 레이아웃(목록/설정/미리보기)으로 가독성 및 조작성 대폭 향상
- **[파일 목록 관리]** 추가된 파일들을 리스트로 확인하고, 리스트 내에서 **더블 클릭하여 실제 파일의 이름을 즉시 수정**하는 기능 추가
- **[드래그 앤 드롭]** `tkinterdnd2`를 통합하여 윈도우 탐색기에서 앱으로 직접 파일이나 폴더를 던져 넣는 기능 추가
- **[캔버스 확장]** 비좁았던 결과물 캔버스 크기를 원본 해상도의 2배로 강제 확장하여 엔진 내 작업 여유 공간 대폭 확보

## 📥 설치 및 실행 방법

1. **사전 준비:** 시스템에 `Python 3.x` 가 설치되어 있어야 합니다.
2. **실행:**
   - 프로젝트 폴더 내에 있는 **`run.bat`** 파일을 더블 클릭합니다.
   - 최초 실행 시 필요한 패키지(`Pillow`, `tkinterdnd2`)가 자동으로 다운로드 및 설치된 후 바로 UI가 열립니다.

## 💡 사용 팁
- **파일 이름 변경:** 좌측 파일 목록 리스트에서 파일 이름을 **더블 클릭**하면 팝업이 뜹니다. 여기서 이름을 바꾸면 실제 윈도우에 있는 파일 이름도 함께 변경됩니다.
- **덮어쓰기 모드:** 대량의 애니메이션 프레임을 덮어쓸 때는 '원본 파일 덮어쓰기' 체크 후 진행하면 작업 폴더가 훨씬 깔끔해집니다. (되돌리기가 안되니 덮어쓰기 전 미리보기에서 꼭 확인하세요!)
