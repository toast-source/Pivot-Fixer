# PNG 피봇 보정 툴 (Pivot Fixer)

이 프로그램은 PNG 이미지의 보이는 영역(알파 채널 기준)을 계산하여 캔버스 크기를 투명하게 확장함으로써, 이미지의 **"피봇(중심)"**을 자동으로 보정해 주는 데스크톱 애플리케이션입니다.

## 주요 기능

1. **세로 피봇 보정 (0~6px):** 
   - 최종 캔버스의 중심을 기준으로 실제 보이는 이미지의 중심이 정확히 N픽셀 아래로 오도록 투명 캔버스를 확장합니다.
2. **좌우 피봇 보정 (ON/OFF):**
   - **ON:** 보이는 이미지의 중심을 최종 캔버스의 가로 중앙에 정확히 맞춥니다.
   - **OFF:** 원본 이미지의 좌우 캔버스 여백 관계를 그대로 유지하며 불필요하게 좌우로 이미지를 이동시키지 않습니다.
3. **일괄 처리 기능:**
   - 폴더 단위로 수많은 PNG 파일을 한 번에 처리할 수 있습니다.
4. **미리보기:**
   - 설정값을 변경할 때마다 즉시 처리된 이미지를 십자선(최종 캔버스 중심)과 함께 미리 볼 수 있습니다.
5. **안전성:**
   - 원본 파일을 덮어쓰지 않고 `파일명_pivotfix.png` 형식으로 새로 저장합니다. (이미지 데이터나 픽셀을 자르는 일 없이 항상 보존합니다)

## 실행 방법

1. **설치 환경:** Python 3.x 가 시스템에 설치되어 있어야 합니다.
2. **실행:** 폴더 내에 있는 `run.bat` 파일을 더블 클릭하여 실행합니다. (필요한 패키지가 자동으로 설치된 후 UI가 열립니다.)

## 사용 방법

1. **파일/폴더 열기:** 좌측 메뉴에서 수정할 단일 PNG 파일이나 PNG 파일들이 있는 폴더를 선택합니다.
2. **설정:** 세로 보정값과 좌우 보정 여부를 선택합니다. 가운데와 우측 패널에서 결과물을 미리 확인할 수 있습니다.
3. **출력 위치:** 기본적으로 원본 파일이 있는 폴더에 저장됩니다. 원할 경우 따로 저장될 폴더를 지정할 수 있습니다.
4. **처리 시작:** 하단의 '처리 시작' 버튼을 누르면 작업이 완료되고 완료된 파일 개수가 표시됩니다.
